Themer is your one stop tool for changing Themes

feren OS uses 'feren' (Based on Arc) as the default theme package. It uses the 'feren' theme for Gtk, windows, panels, login window and lock screen and 'Inspire' as the icon theme.

Now, you may be confused about how to use feren OS, and so, to fix that, we created Themer which is basically a program which is capable of changing the whole DE in one click to look as close to that selected OS and/or theme as possible.

As feren OS values the freedom of users, we provide easy options to switch between our default feren OS style and anything else you want.
