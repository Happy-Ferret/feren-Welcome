‌Recommended Applications‌
These Applications are recommended, for their own reasons as described below, that used to be installed by default in feren OS, but were removed due to security reasons, or ISO Size reasons.

Variety

Variety is a Wallpaper Slideshow Utility that you can check out. Variety is capable of downloading wallpapers from chosen sources, configured to be anti-NSFW by default in feren OS. This used to be in feren OS, but was removed due to not too many people using it, and so it being left as wasted space.

WINE

WINE is a Must Have for people who have Windows Applications, as this application lets you run Windows Applications from within feren OS without an emulator, or Windows Product Key needed to be used. But be aware, with this power, comes the risk of being infected with Windows Viruses, hence, its removal by default. PlayOnLinux is installed too because it provides easy Application Installation for Applications known to be finicky.

‌Remove‌   ‌Install WINE & PlayOnLinux‌  ‌Applying Changes...‌

uGet

uGet is a Download Manager, that's feature packed, yet simple enough to use at the same time. One of the best features for me is the ability to increase the number of Connections at a time to download a file, which means the Download is done faster, at the potential cost of bandwidth. This has never been included with feren OS, yet.

Quadrapassel

A Tetris Wannabe built to work natively for feren OS, and designed to fit in with all of feren OS. Games weren't really added into feren OS, sadly though.

Games Pack

A pack of Games, that again weren't ever included in feren OS, to suit one's boredom, and also replace the suite of games Windows used to include back in the days when Windows was good.
